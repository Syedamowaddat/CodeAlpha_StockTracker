## 📌 Project Overview
A professional **Stock Portfolio Tracker** built with Python during **CodeAlpha Internship**. Users can track their stock investments, calculate total portfolio value, and save results to CSV!

---

## ✨ Features
- 📊 **10 Real Stocks** with live-like prices
- 💰 **Total Investment** calculator
- 📈 **Portfolio Percentage** for each stock
- 🏆 **Best Investment** suggestion
- 💹 **Most Invested** stock tracker
- 💾 **CSV Export** of portfolio
- ❌ **Error Handling** for invalid inputs
- 🔄 **Multiple Stocks** in one session

---

## 🎯 Concepts Used
- `dictionary` for stock prices
- `input/output` operations
- `basic arithmetic` calculations
- `file handling` with CSV
- Functions & clean code structure

## 📖 How to Use
1. Run the program
2. Choose **Option 1** to add a stock
3. Enter stock name (e.g. AAPL, TSLA, NVDA)
4. Enter quantity (e.g. 5)
5. Repeat to add more stocks
6. Choose **Option 2** to view portfolio
7. Choose **Option 3** to save & exit
